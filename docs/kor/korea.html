📱 AroundUs 커뮤니티 앱

프로젝트 세부 정보 확인: AroundUs 커뮤니티 앱

지역 커뮤니티를 연결하는 모바일 애플리케이션을 개발하였습니다. 주요 기능으로는 실시간 채팅, 이벤트 조직, 위치 기반 서비스가 포함됩니다. Node.js와 Docker를 사용하여 마이크로서비스 아키텍처를 구현하고, CI/CD 파이프라인으로 원활한 배포를 지원하였습니다. React Native를 활용하여 iOS와 Android에서 모두 원활하게 작동하는 크로스 플랫폼 모바일 개발을 완료하였습니다. RESTful API를 사용하여 확장 가능한 백엔드 시스템을 설계하고, 효율적인 데이터 관리와 사용자 상호작용을 가능하게 하였습니다. 지도, 알림, 소셜 미디어 공유 등 기능 강화를 위해 서드파티 API를 통합하였습니다.
